linux使用命令记录

ubuntu centos
linux
发布日期:   2019-10-13
更新日期:   2020-06-01
阅读次数:  

1.结束进行中的服务

​ 先查看进行中的服务：sudo netstat -tupln
​ 杀死进程：sudo kill -9 pid(pid号)

2.查找某个文件的位置

​ sudo find / -name file-name (这个是模糊查询，只要位置上有这个名字的都可以)
​ whereis redis (查找某个程序安装的位置，程序名必须正确，不然找不到)
​ which redis （查看执行文件的位置）

3.检查某个服务有没有被占用（例如redis）

​ ps -ef | grep redis
​ sudo netstat -nltp|grep 2181 //查看2181端口的进程

4.手动开启deluge(使用端口8112,默认密码deluge)

deluged
deluge-web –fork

5 docker建立redis容器并用特定配置文件启动

​ docker run -d -p 6379:6379 -v /usr/redis/redis.conf:/etc/redis/redis.conf -v /usr/redis/data:/data –name myredis redis:latest
​ docker中无法使用中文，在进入容器时键入一下命令：docker exec -it 容器名称 env LANG=C.UTF-8 bash //注意LANG=C.UTF-8大写，不要小写
5.1简单安装redis,使用如下命令：
​ (1).tar -zxvf redis-5.0.7.tar.gz
​ (2).cp redis-5.0.7 /home/glh
​ (3).cd redis-5.0.7
​ (4).make //这一步会出错，原因是缺少gcc、gcc-c++，自行安装就行
​ (5).sudo make install
​ (6).进入redis-cli客户端命令行模式之后可以输入shutdown关闭服务端，然后退出客户端就行了;
​ 或者直接在命令行输入一下命令：redis-cli shutdown；但是要是设置了密码，好像就关不了，还是先使用ps -ef | grep redis-server，然后使用kill -9 pid号结束redis进程。
5.2启动redis
直接在命令行输入就行：
　　　(1).redis-server
(2).以特定的配置文件启动：redis-server ./redis.conf
(3).默认启动redis客户端：redis-cli
(4).访问特定的ip的redis服务器：redis-cli -h 127.0.0.1 -p 6379
(5).redis设置密码：编辑redis.conf中的requirepass字段，设置为requirepass 123456，也就是密码为123456
(6).默认16个数据库，下标从0开始，使用select index来选择数据库，例如select 1;选择第二个数据库
5.3.带密码启动redis
先使用redis-cli进入redis，使用ping进行测试是否安装成功,返回pong成功,如果设置了密码，就使用如下的代码进行密码验证
auth 密码
5.4.使用自定义快捷键Ctrl+F2打开redis-desktop-manager

6.启动ngrok:

​ ./bin/ngrokd -domain=”自定义的域名” -httpAddr=”:端口号”

7.docker

刚开始启动前端管理界面：docker run -d --name portainerUI -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock portainer/portainer

7.0.0 进入docker中的MySQL容器： docker exec -it centos-mysql(mysql容器名字) bash

7.0 查看docker版本：sudo docker version

7.1 启动/关闭/重启容器：sudo docker start/stop/restart 容器名字或者ID   //图形化界面账号及密码admin  12345678
7.2 查看正在运行的容器： sudo docker ps
7.3 查看所有的容器：sudo docker ps -a
7.4 删除容器：sudo docker rm -f 容器名字或者ID

7.5 查看已有镜像：sudo docker images
7.6 删除镜像：sudo docker rmi -f 镜像名字或者ID 

7.7 查看镜像列表：sudo docker search 镜像名字:版本号  //具体到官网https://hub.docker.com/查版本

7.8 进入容器：sudo docker exec -it 容器ID /bin/bash
7.9 动态查看容器日志：sudo docker logs 容器名字或者ID

8.打开微信开发者工具

下载：git clone https://github.com/cytle/wechat_web_devtools.git -depth 1
安装：cd wechat_web_devtools
      # 自动下载最新 `nw.js` , 同时部署目录 `~/.config/wechat_web_devtools/`
      ./bin/wxdt install
运行可视化界面：    ./bin/wxdt

9.解压zip文件，防止乱码

法一： unzip -O gbk xxx.zip //记住O要大写才行
法二： sudo apt install unar
#然后使用命令解压文件
unar file.zip
