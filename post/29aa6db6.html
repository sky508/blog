Github Pages设置自定义域名

发表于 2020-05-08 | 分类于 技巧介绍

一、准备域名

我是在阿里云购买的，链接https://www.aliyun.com/

这中间得准备一下注册人信息什么的，这个根据网页指示进行就可以了。

二、解析

点击这个域名，进入域名列表界面，然后点击进入域名解析列表

进入后点击解析设置

进入点击添加记录

1处：选A就行

2处：就是你域名的前缀，我想要的是www.glh508.top和glh508.top，所以我此处第一次是www，第二次不填就行

3处：默认即可

4处：填ip，这个ip怎么获取呢？比如说我github pages开启之后的地址是https://sky508.github.io/blog，所以在cmd中ping一下 sky508.github.io就行(请求超时没事)，填下面那个ip地址就行

如下图：

最后显示就是这个红圈圈住的

三、github设置

在setting中找到此处，

1处：填你的域名

2处：勾选上，安全一些

Save之后需要等待一会能访问到，因为勾上2处之后他会去生成ssl证书，只要出现3处绿色情况就行，说明可以了，然后点一下你就可以发现可以访问你的静态博客页面了
